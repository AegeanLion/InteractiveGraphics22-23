Gameboy Scene

https://aegeanlion.github.io/InteractiveGraphics22-23/
This project is a scene made in Gameboy Studio. It can run on a gameboy. Making this was a very tedious process as Gameboy Studio is not super user friendly and it is very limiting. In the end, we were able to produce a mostly functioning game where you try to destroy asteroids that fly around and try to not let them hit the front of your ship.
